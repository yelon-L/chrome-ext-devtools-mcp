## 🔄 文档维护流程

### 1. 新增文档

- 确定文档类别
- 添加到本索引
- 更新相关链接
- 标注重要性

### 2. 更新文档

- 更新文档内容
- 更新"最后更新"日期
- 更新版本号（如适用）

### 3. 归档文档

- 移动到 `docs/archive/`
- 从索引中移除或标注"已归档"
- 添加归档原因

### 4. 定期审查

- 每个版本发布前审查所有文档
- 标注过时文档
- 合并重复文档
- 更新示例和截图
